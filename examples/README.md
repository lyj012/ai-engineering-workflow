# Examples

This directory contains public, sanitized examples for the two-step workflow chain.

- `minimal-target/`: a tiny shell project that can be used as the target repository.
- `requirements/simple-greeting.md`: a copyable user requirement.
- `artifacts/plan-ready/`: sample output from `plan-from-requirement`.
- `artifacts/delivery-success/`: sample output from `deliver-from-plan`.

The examples are intentionally small so readers can inspect the full chain without customer code or local machine paths.
